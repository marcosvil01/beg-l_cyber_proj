# 🔐 CyberHub: Ultimate Master Suite (v2.0)

CyberHub is a professional-grade, modular cybersecurity education and testing platform. It integrates **25 specialized tools** into a unified, glassmorphic dashboard built with Python and CustomTkinter.

> [!WARNING]
> **LEGAL DISCLAIMER**: Use this tool ONLY for educational purposes and testing on networks/systems you own. Unauthorized use is illegal.

---

## 🚀 25 Projects & Modules

CyberHub is organized into four distinct security phases:

### ⚔️ Offensive (Red Team)
1. **Multi-Port Honeypot**: Decoy service with advanced logging.
2. **Password Cracker**: Multi-threaded dictionary and brute-force modes.
3. **Network Sniffer**: Real-time packet capture with protocol filters.
4. **Keylogger**: Stealthy keystroke monitoring for training.
5. **Wi-Fi Scanner**: WLAN discovery and security auditing.
6. **Network Port Scanner**: High-speed discovery of open services.
7. **Phishing Simulator**: Security awareness training with HTML templates.
8. **GPU Crack Theory**: Educational benchmarks for parallel hashing.

### 🛡️ Defensive (Blue Team)
9. **AES-256 Encryption**: Secure file locker with key management.
10. **2FA Authenticator Hub**: Persistent TOTP manager (JSON-backed).
11. **Firewall Manager**: Rule generation for Windows advanced firewall.
12. **Snort Log Parser**: Deep analysis of IDS/IPS alerts.
13. **DNS & ARP Monitor**: Detection of spoofing and network anomalies.
14. **Antivirus Sim**: Signature-based scanning against a threat database.
15. **TLS/SSL Explorer**: Real-time probe of domain certificates.

### 🔍 Forensics & Analysis
16. **Malware Analyzer**: Static analysis (entropy, strings, indicators).
17. **Vuln Scanner**: Windows security configuration audit.
18. **Metadata Extractor**: EXIF data recovery from digital media.
19. **Activity Monitor**: Real-time system monitoring (CPU/RAM/Processes).
20. **Zero-Day Research DB**: Historical repository of famous exploits.

### 🌐 Web & Lab
21. **Secure Web App**: Flask-based demo of SQLi and XSS vulnerabilities.
22. **Tor Proxy Hub**: Simulated circuit routing and documentation.
23. **Safety Browser Extension**: Security-focused template for Chrome.
24. **Cyber Lab Setup Guide**: Master guide for VirtualBox and Kali.
25. **Decoy Assets**: Advanced honeypot interaction simulation.

## 🛠️ Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/beg-l_cyber_proj.git
cd beg-l_cyber_proj

# Install dependencies
pip install -r requirements.txt
```

## 📦 Requirements
- `customtkinter`: Modern UI framework.
- `psutil`: System metrics.
- `scapy`: Packet manipulation.
- `cryptography`: AES encryption.
- `pyotp`: 2FA logic.
- `flask`: Web demo server.
- `pillow`: Image processing (metadata).

## 🧭 How to Use
1. Run `python main.py`.
2. Browse categories via the sidebar.
3. Use the **integrated terminal** to monitor actions in real-time.
4. Launch any of the 25 projects; each will open in a self-contained, focus-locked window.
